# 倉庫番 📦

為小學生高年級設計的推箱子小遊戲，繁體中文介面、**50 個由淺入深的關卡**（依最少推動次數排序，5 推 → 68 推），鍵盤與觸控皆可操作。

**關卡來源**：全數採用 David W. Skinner 的 [Microban](http://www.sourcecode.se/sokoban/levels) I & II 公有（public domain）題庫，並經 BFS solver 驗證每關都可解、標註最少推動次數。

## 線上試玩

👉 <https://shinychang.github.io/sokoban/>

## 怎麼玩

- 方向鍵 `↑` `↓` `←` `→` 或 `WASD` 移動
- `Z` 上一步（悔棋）
- `R` 重新開始本關
- 把所有箱子（📦）推到目標位置（🎯），箱子就位後會變成 ✅

箱子只能「推」不能「拉」。撞到牆或另一顆箱子就推不動。

## 特色

- **50 個關卡**：依最少推動次數排序（5 → 68 推），難度平滑遞進
- **難度四階**：簡單、中等、困難、挑戰，每關標註最少推動次數
- **完美解答識別**：達到最少推動次數會顯示 ✨ 標記
- **進度自動儲存**：用瀏覽器 LocalStorage 保存通關紀錄與最佳步數
- **隨時悔棋**：步驟紀錄完整，按幾次 `Z` 就能退回
- **平板 / 手機友善**：永遠顯示觸控方向鍵
- **純前端**：零相依套件，純 HTML / CSS / JS

## 本地執行

任意靜態伺服器即可，例如：

```bash
cd sokoban
python3 -m http.server 8080
# 開 http://localhost:8080
```

## 檔案結構

```
sokoban/
├── index.html    # 主頁面與各個畫面
├── style.css     # 視覺樣式
├── game.js       # 遊戲核心邏輯
└── levels.js     # 關卡資料
```

## 新增關卡

在 `levels.js` 的 `LEVELS` 陣列加一筆：

```js
{
  name: "第 N 關 · 副標題",
  difficulty: "easy", // easy / medium / hard / expert
  map: [
    "#####",
    "# . #",
    "# $ #",
    "# @ #",
    "#####",
  ],
},
```

符號說明：`#` 牆、` ` 地板、`.` 目標、`$` 箱子、`*` 箱子在目標上、`@` 玩家、`+` 玩家在目標上。
